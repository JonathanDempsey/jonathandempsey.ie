The deforestation website was created to teach people about deforestation in Africa, and what you can do to help stop it.

The site uses basic HTML to create the sites structure and basic CSS to design it. All assets, excluding the YouTube video, were created using Photoshop and Flash. The photo gallery is implemented using slide.js. The site was made from a combination of all classes I was taking in first year. I learned research skills from looking up information about deforestation in Africa and finding credible references. The images and logo were created in using basic Photoshop skills. An animation about deforestation was made using Flash. Basic HTML and CSS was then used to make the site and all the other resources were implemented to create the websites content.
